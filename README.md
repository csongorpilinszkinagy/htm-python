## The HTM network
Hierarchical Temporal Memory (HTM) is a type of neural network that is modeled after the structure and function of the human neocortex. The neocortex is the part of the brain responsible for many of our higher cognitive functions, such as perception, language, and reasoning. The HTM network is designed to perform tasks such as pattern recognition, anomaly detection, and prediction, among others. There are several advantages to using HTM networks over traditional neural networks. One of the main advantages is that HTM networks are highly scalable and can handle large and complex datasets with ease. Additionally, they are adaptive and can learn in real-time, making them ideal for applications that require continuous learning and adaptation. HTM networks are also robust to noise and can perform well even when the data is noisy or incomplete. Finally, HTM networks are highly interpretable, which means that they can provide insights into how they arrive at their decisions, making them useful for tasks where transparency and explainability are important.

## Sparse Distributed Representation

Sparse Distributed Representation (SDR) is a fundamental concept in the HTM network. It is used to represent input data as a binary vector with a small number of active bits and a large number of inactive bits. The SDR representation is designed to capture the sparsity and redundancy in the input data, allowing the network to efficiently and accurately process and analyze large and complex datasets. In the HTM network, SDRs are generated by the SDR encoder, which maps the input data onto a high-dimensional space, where each dimension corresponds to a feature or attribute of the input. The SDR encoder then identifies which dimensions are active (or "on") for a given input, resulting in a sparse binary representation. SDRs are used throughout the HTM network, including in the Temporal Memory (TM) layer, where they represent the activity of neurons and synapses. SDRs allow the HTM network to efficiently learn and recognize patterns and sequences in the input data, while minimizing redundancy and noise.

## Hebbian learning

Hebbian learning is used to adapt the connections between neurons in the network based on their co-activation. This type of learning is used to learn the temporal and spatial patterns in the input data and to build a hierarchical representation of the data. In the HTM network, the Hebbian learning rule is applied at the level of the synapse between two neurons, and it works by modifying the strength of the connection between them based on their level of co-activation. Specifically, if two neurons are active at the same time, the connection between them is strengthened, while if they are not active together, the connection is weakened.

## Synapses

Synapses are the connections between neurons in the Hierarchical Temporal Memory (HTM) network. Synapses are responsible for transmitting signals between neurons and are critical for the function of the HTM network. In the HTM network, synapses are modeled as a set of binary values that represent the strength of the connection between two neurons. These binary values are called synaptic weights and are used to determine the influence of one neuron on another. The synaptic weights are updated using Hebbian learning, which modifies the strength of the connections between neurons based on their co-activation. The strength of the synapses in the HTM network is essential for the network's ability to learn and recognize patterns in the input data.

## Segments

Segments are groups of synapses that form a connection between a subset of neurons in the Temporal Memory (TM) layer. These synapses are grouped together to represent a specific pattern or sequence in the input data. During the training phase, the TM layer creates new segments by selecting a group of synapses that are active and connected to the same set of neurons. The synapses in the segment are then strengthened through Hebbian learning, allowing the segment to represent a specific pattern or sequence in the input data. The segments allow the TM layer to learn and store patterns in the temporal context of the input data. The segments also provide a mechanism for the TM layer to activate multiple neurons simultaneously, allowing the network to represent complex (multiple) patterns and sequences.

### SDR encoder

Key features of an SDR:
* Sparse binary vector
* Similarity is retained by overlapping bits

Sparse Distributed Representation (SDR) encoder is a key component of the Hierarchical Temporal Memory (HTM) network. The SDR encoder is responsible for transforming the input data into a sparse binary representation, which is then processed by the HTM network. The SDR encoder works by mapping the input data onto a high-dimensional space, where each dimension corresponds to a feature or attribute of the input. The SDR encoder then identifies which dimensions are active, or "on," for a given input. By representing the input data in a sparse binary form, the SDR encoder can effectively capture the underlying patterns and relationships in the data, while minimizing redundancy and noise. This sparse binary representation is then passed on to the HTM network for further processing and analysis. The SDR encoder is a critical component of the HTM network, as it enables the network to handle large and complex datasets with high accuracy and efficiency.
